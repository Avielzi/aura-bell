## 🌟 Overview

**Aura Bell** is a cutting-edge, serverless smart doorbell solution designed for the modern home. Built on **Cloudflare Workers**, it offers a robust, high-performance system with **zero hosting costs**. Whether you're looking for a simple notification system or a professional-grade multi-lingual interface with direct calling, Aura Bell has you covered.

---

## 📂 Repository Structure

This repository provides two distinct versions to suit your needs:

| Version | Folder | Description |
| :--- | :--- | :--- |
| **Aura Bell v2.0** | [**`upgraded/`**](./upgraded) | **The Professional Choice.** Features a modern UI, 5-language support, direct calling, and an automated setup script. |
| **Dori-Bell v1.0** | [**`original/`**](./original) | **The Minimalist Template.** A simple, secure serverless doorbell that notifies you via Telegram. |
