# 📖 מדריך למשתמש - Aura Bell v2.0

ברוכים הבאים ל-**Aura Bell**, הגרסה המקצועית והמשודרגת של פעמון הדלת החכם מבוסס ה-Serverless.

## 🌟 מה חדש בגרסה 2.0?
- **מיתוג מקצועי:** שם חדש, לוגו מודרני וממשק משתמש יוקרתי.
- **חיוג ישיר:** אפשרות לאורחים להתקשר אליכם ישירות מהדפדפן.
- **5 שפות:** תמיכה מלאה בעברית, אנגלית, ערבית, רוסית וצרפתית.
- **ממשק הגדרה:** דף הגדרה ייעודי בנתיב `/setup`.

## 🛠️ שלבי התקנה מהירה

### 1. הכנת חשבון Cloudflare
ודאו שיש לכם חשבון ב-Cloudflare (זה בחינם).

### 2. יצירת בוט טלגרם
1. פנו ל-`@BotFather` בטלגרם וצרו בוט חדש.
2. שמרו את ה-**API Token** שקיבלתם.
3. קבלו את ה-**Chat ID** שלכם (ניתן להשתמש בבוטים כמו `@userinfobot`).

### 3. הגדרת Cloudflare Turnstile (הגנת בוטים)
1. בלוח הבקרה של Cloudflare, עברו ל-**Turnstile**.
2. צרו "Site" חדש וקבלו את ה-**Site Key** וה-**Secret Key**.

### 4. הגדרה אוטומטית (מומלץ)
הפעל את סקריפט ההגדרה החדש שיצרנו:
```bash
python3 setup.py
```
הסקריפט יבקש ממך את כל הנתונים ויעדכן את הקבצים אוטומטית.

### 5. פריסת הקוד
1. פתחו את הקובץ `worker.js`.
2. העתיקו את כל תוכנו ל-Worker חדש ב-Cloudflare.
3. תחת לשונית **Settings -> Variables**, הגדירו את המשתנים הבאים:
   - `TG_BOT_TOKEN`: הטוקן מהטלגרם.
   - `TG_CHAT_ID`: המזהה מהטלגרם.
   - `TURNSTILE_SITE_KEY`: המפתח הציבורי של Turnstile.
   - `TURNSTILE_SECRET`: המפתח הסודי של Turnstile.
   - `PHONE_NUMBER`: מספר הטלפון שלכם (בפורמט בינלאומי, למשל `+972...`).
   - `FAMILY_NAME`: השם שיוצג על הפעמון.

### 5. שימוש בממשק ההגדרה
לאחר הפריסה, תוכלו לגשת לכתובת ה-Worker שלכם ולהוסיף `/setup` בסוף כדי לראות את המדריך הגרפי.
