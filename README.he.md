# 🔔 Aura Bell v2.0: פעמון דלת חכם ומקצועי

> **Aura Bell** היא מערכת פעמון דלת חכמה, רב-לשונית ומאובטחת. היא מספקת התראות טלגרם מיידיות, אפשרות לחיוג ישיר וממשק מודרני ויוקרתי – הכל פועל בעלות של $0 על גבי Cloudflare Workers.

---

## 🌍 תמיכה ב-5 שפות
Aura Bell תוכננה לקהל גלובלי עם תמיכה מלאה ב:
- 🇮🇱 **עברית**
- 🇺🇸 **אנגלית**
- 🇸🇦 **ערבית**
- 🇷🇺 **רוסית**
- 🇫🇷 **צרפתית**

## 🚀 התקנה מהירה (בחר את הדרך שלך)

### אפשרות א': הדרך הקלה (לכולם)
1. **הורד** את קבצי הפרויקט.
2. **הפעל הגדרה:** פתח את הטרמינל והרצ:
   ```bash
   python3 setup.py
   ```
   *הסקריפט ידריך אותך צעד אחר צעד.*
3. **פריסה:** הרץ `wrangler deploy` או העתק את תוכן `worker.js` ל-Worker חדש ב-Cloudflare.

### אפשרות ב': למפתחים
1. **שכפול והתקנה:**
   ```bash
   git clone https://github.com/Avielzi/dori-bell-serverless-template.git aura-bell
   cd aura-bell
   npm install -g wrangler
   ```
2. **הגדרה:** ערוך את `wrangler.toml` או הגדר משתני סביבה בלוח הבקרה של Cloudflare.
3. **פריסה:** `wrangler deploy`
