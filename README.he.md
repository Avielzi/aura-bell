## 🤯 למה Aura Bell?

בעידן של בתים חכמים, פעמוני דלת פיזיים הם לרוב רועשים ומוגבלים. **Aura Bell** מספקת פתרון מקצועי ללא שרת שהוא חכם יותר, שקט יותר, וחינמי לחלוטין להפעלה. זהו השילוב המושלם בין ארכיטקטורת ענן מודרנית לבין עיצוב ממוקד משתמש.

## 🚀 התקנה מהירה

1. **שכפול ופריסה:**
   ```bash
   git clone https://github.com/Avielzi/dori-bell-serverless-template.git aura-bell
   cd aura-bell
   wrangler deploy
   ```
2. **הגדרה:** בקרו בכתובת `your-worker-url.workers.dev/setup` למדריך ההגדרה.
3. **משתני סביבה:** הגדירו את `TG_BOT_TOKEN`, `TG_CHAT_ID`, ו-`PHONE_NUMBER` בלוח הבקרה של Cloudflare.
